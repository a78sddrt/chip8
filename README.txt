# CHIP-8 Emulator (Python)

This is a CHIP-8 emulator written in Python.

## ROM Sources

The test and game ROMs used are from:

1. [Timendus/chip8-test-suite](https://github.com/Timendus/chip8-test-suite)
2. [kripod/chip8-roms](https://github.com/kripod/chip8-roms)

## How to Run

### Option 1: Run the App on macOS
Navigate to the `dist/` directory and double-click `chip8.app`.

### Option 2: Run from Terminal
In the `dist/` directory, you can also run the compiled binary (`chip8`) via terminal.

### Option 3: Run the Python script directly
You can also run the script with Python:

```bash
python chip8.py
